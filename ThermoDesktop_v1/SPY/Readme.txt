Программа, отображающая температуру с проекта ThermoDisplay4х7 на компьютере.
Состоит из:
1) Модуль, подключенный по USB к компьютеру (платформа СДТ). Прошивка - в папке RfNodesDetector_Module
2) Десктопная программа для Windows. dot NET-framework 4.1, папка RfNodeSpy_Program. Отображает температуру на улице, дома, время последнего опроса, а так же сохраняет данные о температуре после каждого запроса в файл C:\Log.csv
